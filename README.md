# SDL3 C# Ported Examples

A collection of C# examples ported from the official SDL3 repository.
This project is designed to help developers learn SDL3 using the `SDL3-CS` wrapper.

## Prerequisites
- .NET 10 SDK installed on your machine.
- Git installed.

## Installation
To compile all examples at once:
1. Clone this repository.
2. Run `dotnet publish` in the root directory.
4. The executable files will be generated in the output directory Build

## Technical Specifications
- **Wrapper:** [SDL3-CS](https://github.com/edwardgushchin/SDL3-CS)
- **Original Source:** [SDL3 Examples](https://github.com/libsdl-org/SDL/tree/main/examples)
- **Native Library:** SDL3 v3.4.12 (Binaries for Windows and Linux are bundled).
